## Начало работы

### Вход в систему

1. Откройте браузер и перейдите по адресу `http://localhost:5000`
2. Вы будете перенаправлены на страницу входа
3. Введите учетные данные:
   - **Администратор**: `admin` / `admin123`
   - **Эксперт**: используйте учетные данные, предоставленные администратором

### Интерфейс

После входа вы увидите:
- Навигационное меню вверху страницы
- Информацию о текущем пользователе и роли
- Кнопку выхода из системы

## Работа администратора

### Управление пользователями

#### Создание пользователя-эксперта

1. Перейдите в раздел **"Пользователи"**
2. Нажмите **"Добавить пользователя"**
3. Заполните форму:
   - **Имя пользователя**: уникальное имя для входа
   - **Пароль**: безопасный пароль
   - **Роль**: выберите "Эксперт"
   - **Эксперт**: выберите эксперта из списка (сначала создайте эксперта)
4. Нажмите **"Создать"**

#### Создание эксперта

1. Перейдите в раздел **"Эксперты"**
2. Нажмите **"Добавить эксперта"**
3. Введите имя эксперта
4. Нажмите **"Сохранить"**

**Важно**: После создания эксперта создайте для него пользователя в разделе "Пользователи".

### Управление активами

#### Добавление актива

1. Перейдите в раздел **"Активы"**
2. Нажмите **"Добавить актив"**
3. Введите название актива
4. Нажмите **"Сохранить"**

После добавления актива эксперты смогут его оценивать.

#### Редактирование актива

1. В списке активов нажмите **"Редактировать"** рядом с нужным активом
2. Измените название
3. Нажмите **"Сохранить"**

### Просмотр оценок

Администратор видит **все оценки** всех экспертов:
- В разделе **"Оценки активов"** - все оценки по критериям
- В разделе **"Вероятности угроз"** - все оценки вероятностей

### Анализ рисков

1. Перейдите в раздел **"Анализ рисков"**
2. Нажмите **"Добавить запись анализа рисков"**
3. Заполните форму:
   - Выберите актив, владельца, угрозу, уязвимость
   - Выберите принятые меры
   - Если выбрана "Минимизация рисков и выбор контролей", укажите меру контроля
   - Укажите эффективность защиты (0-1)
4. Нажмите **"Добавить"**

### Просмотр критичности и рисков

1. Перейдите в раздел **"Критичность и риски"**
2. Вы увидите:
   - Веса критериев
   - Таблицу ранжирования активов с расчетом рисков
   - Тепловую карту остаточных рисков
   - Детальную таблицу анализа рисков

## Работа эксперта

### Оценка активов

#### Добавление оценки актива

1. Перейдите в раздел **"Оценки активов"**
2. Нажмите **"Добавить оценку актива"**
3. Заполните форму:
   - Выберите актив из списка
   - Эксперт выбран автоматически (ваше имя)
   - Оцените актив по каждому критерию (0-10):
     - Жизнь/Здоровье
     - Экономика
     - Экология
     - Зависимость
     - Социальное
     - Международное
4. Нажмите **"Сохранить"**

**Примечание**: Каждый эксперт может оценить каждый актив только один раз.

#### Редактирование своей оценки

1. В списке оценок найдите свою оценку
2. Нажмите **"Редактировать"**
3. Измените значения
4. Нажмите **"Сохранить"**

**Важно**: Эксперт может редактировать только свои оценки.

### Оценка вероятностей угроз

#### Добавление вероятности угрозы

1. Перейдите в раздел **"Вероятности угроз"**
2. Нажмите **"Добавить вероятность угрозы"**
3. Заполните форму:
   - Выберите актив
   - Эксперт выбран автоматически
   - Укажите вероятность (1-3):
     - 1 = Низкая вероятность
     - 2 = Средняя вероятность
     - 3 = Высокая вероятность
4. Нажмите **"Сохранить"**

### Просмотр результатов

Эксперт может просматривать:
- Список активов (без возможности редактирования)
- **Только свои** оценки активов
- **Только свои** оценки вероятностей угроз
- Результаты критичности и рисков (для всех активов)

## Оценка активов

### Критерии оценки

Активы оцениваются по 6 критериям (шкала 0-10):

1. **Жизнь/Здоровье** - влияние на жизнь и здоровье людей
2. **Экономика** - экономическое влияние
3. **Экология** - экологическое влияние
4. **Зависимость** - зависимость других систем
5. **Социальное** - социальное влияние
6. **Международное** - международное влияние

### Веса критериев

По умолчанию используются следующие веса:
- Жизнь/Здоровье: 41.9%
- Экономика: 25.2%
- Экология: 9.9%
- Зависимость: 14.4%
- Социальное: 5.1%
- Международное: 3.5%

### Расчет критичности

Критичность = Σ(оценка_критерия × вес_критерия)

### Расчет Impact

Impact = 1 + (критичность / 10) × 2

## Анализ рисков

### Вероятности угроз

Эксперты оценивают вероятность угроз по шкале 1-3:
- **1** = Низкая вероятность
- **2** = Средняя вероятность
- **3** = Высокая вероятность

Система автоматически усредняет оценки всех экспертов.

### Расчет риска

1. **Первоначальный риск** = Impact × Вероятность
2. **Остаточный риск** = Первоначальный риск × (1 - Эффективность_контроля)

### Уровни риска

- **Низкий** (1.0-3.9): Допустимый, контрольный
- **Средний** (4.0-6.9): Требует мер по снижению
- **Высокий** (7.0-9.0): Требует приоритетного устранения

### Принятые меры

Доступны 4 типа мер:
1. Минимизация рисков и выбор контролей
2. Передача рисков третьей стороне (страхование)
3. Отказ от риска
4. Принятие риска

### Меры контроля

Система поддерживает меры контроля по стандарту ISO 27001 (A.5.1.1 - A.18.2.3).

## Советы по использованию

1. **Для администратора**:
   - Сначала создайте экспертов, затем пользователей для них
   - Регулярно проверяйте оценки экспертов
   - Используйте анализ рисков для принятия решений

2. **Для эксперта**:
   - Оценивайте активы объективно
   - Используйте всю шкалу (0-10) для более точных оценок
   - Регулярно проверяйте свои оценки

3. **Общие рекомендации**:
   - Сохраняйте резервные копии базы данных
   - Регулярно обновляйте оценки при изменении обстоятельств
   - Используйте тепловые карты для визуального анализа рисков

## Часто задаваемые вопросы

**Q: Могу ли я изменить оценку после сохранения?**  
A: Да, вы можете редактировать свои оценки через кнопку "Редактировать".

**Q: Что делать, если я забыл пароль?**  
A: Обратитесь к администратору для сброса пароля.

**Q: Почему я не вижу некоторые разделы меню?**  
A: Некоторые разделы доступны только администратору.

**Q: Как интерпретировать тепловую карту?**  
A: Зеленый цвет = низкий риск, желтый = средний, красный = высокий.

**Q: Можно ли экспортировать данные?**  
A: В текущей версии экспорт не реализован, но данные доступны в базе SQLite.
