# История изменений

## [1.1.0] - 2024

### Добавлено
- Система аутентификации с ролями (администратор/эксперт)
- Управление пользователями
- Разграничение доступа к данным
- Страница входа/выхода
- Декораторы для контроля доступа
- Эксперты видят только свои оценки
- Обновленная навигация с информацией о пользователе

### Изменено
- Все маршруты защищены декораторами доступа
- Формы оценок адаптированы для экспертов
- Навигационное меню скрывает недоступные разделы

### Безопасность
- Хеширование паролей
- Сессии для аутентификации
- Проверка прав доступа при операциях

---

## [1.0.0] - 2024

### Добавлено
- Управление активами
- Управление экспертами
- Экспертные оценки активов по 6 критериям
- Оценка вероятностей угроз
- Расчет критичности активов
- Расчет рисков (первоначальный и остаточный)
- Классификация рисков (низкий/средний/высокий)
- Тепловые карты остаточных рисков
- Анализ рисков с мерами контроля
- Справочники угроз, уязвимостей, мер контроля
- Визуализация результатов

### Функциональность
- CRUD операции для всех сущностей
- Автоматический пересчет средних оценок
- Взвешенный расчет критичности
- Интеграция с ISO 27001 (меры контроля)

### Технологии
- Flask веб-фреймворк
- SQLite база данных
- Matplotlib/Seaborn для визуализации
- Tailwind CSS для интерфейса
